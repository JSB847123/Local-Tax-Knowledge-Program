# Tax-Flow 로컬 프로그램

`prd.md`의 핵심 요구사항을 바탕으로 만든 로컬 실행형 Tax-Flow 프로그램입니다. Python만 있으면 로컬 서버를 켜고 브라우저에서 바로 사용할 수 있습니다.

## 실행

### 권장 실행

1. `start-taxflow.bat`을 더블클릭합니다.
2. 브라우저가 자동으로 열리면 Tax-Flow를 사용합니다.
3. 프로그램을 종료하려면 열린 콘솔 창에서 `Ctrl+C`를 누릅니다.

### 명령어 실행

```powershell
python run_taxflow.py
```

기본 주소는 `http://127.0.0.1:8765/`입니다. 해당 포트가 사용 중이면 가까운 빈 포트를 자동으로 사용합니다.

### 직접 열기

로컬 서버 없이 `index.html`을 브라우저로 직접 열어도 동작합니다. 이 경우 데이터는 브라우저 `localStorage`에만 저장됩니다.

## 저장

- 로컬 서버로 실행하면 데이터가 `data/taxflow-data.json`에 자동 저장됩니다.
- 브라우저 `localStorage`에도 함께 저장되어 기존 직접 실행 방식과 호환됩니다.
- 데이터 백업은 저장 파일 `data/taxflow-data.json`을 보관하는 방식으로 사용할 수 있습니다.

## 구현된 주요 기능

- 세목 워크스페이스 선택
- `법령 및 판례 검색`: 공식 원문 검색 링크와 `LAW_OC` 기반 국가법령정보 API 검색
- `즐겨찾기`: 검색 결과에서 별표한 법령·판례 원문 다시 보기
- `매뉴얼 작성`: `이론`, `관련 법령`, `전산 작업` 3파트 작성
- 로컬 JSON 자동 저장과 브라우저 `localStorage` 호환 저장

## MVP 범위 메모

`LAW_OC`가 설정되지 않은 경우 국가법령정보센터 등은 공식 사이트 검색 링크와 메타데이터 저장 방식으로 동작합니다. `법령 및 판례 검색` 화면에서 `LAW_OC`를 저장하면 로컬 서버가 국가법령정보 검색 API를 호출해 실제 법령·판례·해석례 후보를 먼저 표시합니다.
